Spectrum Sensing for Cognitive Radio using Machine Learning

Background

Spectrum sensing, i.e., detecting the presence of primary users in a licensed spectrum, is a fundamental problem in cognitive radio. Cognitive radio uses a number of technologies including Adaptive Radio (where the communications system monitors and modifies its own performance) and Software Defined Radio (SDR) where traditional hardware components including mixers, modulators and amplifiers have been replaced with intelligent software.

Recon Systems

Another interesting project that I got to work on at Phonepe, were the reconciliation systems. A recon is basically a process that takes place on a daily basis where each service provider sends a file to Phonepe which contains the details of all the transactions that were recorded on their end. This data was matched with the transactions that happened on our(Phonepe) end to make sure that pending transactions are updated. There were basically either of 2 types of Recon set up for each Service Provider - API based and SFTP based. In an API based recon, the statuses of all the pending transactions on our end were checked on service provider's end via API calls, whereas in SFTP based recon the Service provider would upload a file on to an SFTP server which would then be processed by our system in place. I got to design both - SFTP and API based recon systems for Service providers like Visa CyberSource, Airtel, PayU and TPSL. These recon systems cross verified data for millions of transactions on both ends and sent mailing alerts to the stake holders in case of discrepency.
